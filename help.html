Anatomy of an HTML Page

The basic HTML page is divided into head and body sections. The body contains everything shown on the page. The head contains information about the page, but not displayed on it.


<!DOCTYPE html>
<html>
<head>
    <title>Hello, World!</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>

Paragraphs

Paragraph tags wrap each paragraph on your page.


<p>Text inside paragraph 1</p>
<p>Text inside paragraph 2</p>
<p>Text inside paragraph 3</p>

Common Text Formatting Tags

These tags are used to add some style to your text.


<b>This text is bold!</b>
<i>This is italics</i>
<small>This is small</small>
This is the 4<sup>th</sup> line

Ordered List

Use ordered lists to show numbered lists of text.

<ol>
    <li>Pizza</li>
    <li>Tacos</li>
    <li>Hamburgers</li>
</ol>
  1. Pizza
  2. Tacos
  3. Hamburgers

Unordered List

Use unordered lists to group items together, like text on a slide.

<ul>
    <li>Pizza</li>
    <li>Tacos</li>
    <li>Hamburgers</li>
</ul>
  • Pizza
  • Tacos
  • Hamburgers
